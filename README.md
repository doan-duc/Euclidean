# Chương trình Thuật toán Extended Euclidean trong GF(2^10)

Chương trình này thực hiện tìm nghịch đảo nhân của các phần tử trong trường hữu hạn $GF(2^{10})$ sử dụng đa thức tối giản mặc định là $m(x) = x^{10} + x^3 + 1$ (tương ứng với giá trị thập phân 1033).

## Cách hoạt động
1. Chương trình yêu cầu người dùng nhập vào hai giá trị $a$ và $b$ (ở dạng thập phân).
2. Nó sẽ tính toán từng bước của thuật toán Euclidean mở rộng để tìm nghịch đảo của $a$ và $b$ modulo $m(x)$.
3. Kết quả được hiển thị dưới dạng bảng chi tiết các bước tính toán dư ($r$), thương ($q$), và các hệ số ($v, w$).

## Ví dụ chạy thử (Test Vector)
Với đầu vào $a = 523$ và $b = 1015$:

### 1. Tìm nghịch đảo của $a = 523$
Thuật toán dừng lại khi tìm thấy $GCD = 1$. Nghịch đảo nhân của $523$ trong $GF(2^{10})$ là **798**.

### 2. Tìm nghịch đảo của $b = 1015$
Thuật toán dừng lại khi tìm thấy $GCD = 1$. Nghịch đảo nhân của $1015$ trong $GF(2^{10})$ là **709**.

## Chi tiết kết quả tính toán
```text
Euclidean mở rộng cho a = m(x), b = 523 (trong GF(2^10))
m(x) tương ứng giá trị thập phân 1033

+------------------------------------+----------------------------------------+
| Calculate                          | Calculate                              | 
+------------------------------------+----------------------------------------+
| r_-1 = a = 1033                    | v_-1 = 1                               | 
|                                    | w_-1 = 0                               | 
+------------------------------------+----------------------------------------+
| r_0 = b = 523                      | v_0 = 0                                | 
|                                    | w_0 = 1                                | 
+------------------------------------+----------------------------------------+
| r_1 = r_-1 mod r_0                 | v_1 = v_-1 - q_1*v_0                   | 
|      = 1033 mod 523 = 31           |      = 1 ^ (2*0) = 1                   | 
| q_1 = 1033 / 523 = 2               | w_1 = w_-1 - q_1*w_0                   | 
|                                    |      = 0 ^ (2*1) = 2                   | 
+------------------------------------+----------------------------------------+
| r_2 = r_0 mod r_1                  | v_2 = v_0 - q_2*v_1                    | 
|      = 523 mod 31 = 4              |      = 0 ^ (49*1) = 49                 | 
| q_2 = 523 / 31 = 49                | w_2 = w_0 - q_2*w_1                    | 
|                                    |      = 1 ^ (49*2) = 99                 | 
+------------------------------------+----------------------------------------+
| r_3 = r_1 mod r_2                  | v_3 = v_1 - q_3*v_2                    | 
|      = 31 mod 4 = 3                |      = 1 ^ (7*49) = 150                | 
| q_3 = 31 / 4 = 7                   | w_3 = w_1 - q_3*w_2                    | 
|                                    |      = 2 ^ (7*99) = 299                | 
+------------------------------------+----------------------------------------+
| r_4 = r_2 mod r_3                  | v_4 = v_2 - q_4*v_3                    | 
|      = 4 mod 3 = 1                 |      = 49 ^ (3*150) = 395              | 
| q_4 = 4 / 3 = 3                    | w_4 = w_2 - q_4*w_3                    | 
|                                    |      = 99 ^ (3*299) = 798              | 
+------------------------------------+----------------------------------------+
| r_5 = r_3 mod r_4                  | GCD = r_4 = 1                          | 
|      = 3 mod 1 = 0                 | => Nghịch đảo nhân là w_4 = 798        | 
| q_5 = 3 / 1 = 3                    |                                        | 
+------------------------------------+----------------------------------------+

Euclidean mở rộng cho a = m(x), b = 1015 (trong GF(2^10))
m(x) tương ứng giá trị thập phân 1033

+------------------------------------+----------------------------------------+
| Calculate                          | Calculate                              | 
+------------------------------------+----------------------------------------+
| r_-1 = a = 1033                    | v_-1 = 1                               | 
|                                    | w_-1 = 0                               | 
+------------------------------------+----------------------------------------+
| r_0 = b = 1015                     | v_0 = 0                                | 
|                                    | w_0 = 1                                | 
+------------------------------------+----------------------------------------+
| r_1 = r_-1 mod r_0                 | v_1 = v_-1 - q_1*v_0                   | 
|      = 1033 mod 1015 = 16          |      = 1 ^ (3*0) = 1                   | 
| q_1 = 1033 / 1015 = 3              | w_1 = w_-1 - q_1*w_0                   | 
|                                    |      = 0 ^ (3*1) = 3                   | 
+------------------------------------+----------------------------------------+
| r_2 = r_0 mod r_1                  | v_2 = v_0 - q_2*v_1                    | 
|      = 1015 mod 16 = 7             |      = 0 ^ (63*1) = 63                 | 
| q_2 = 1015 / 16 = 63               | w_2 = w_0 - q_2*w_1                    | 
|                                    |      = 1 ^ (63*3) = 64                 | 
+------------------------------------+----------------------------------------+
| r_3 = r_1 mod r_2                  | v_3 = v_1 - q_3*v_2                    | 
|      = 16 mod 7 = 2                |      = 1 ^ (6*63) = 131                | 
| q_3 = 16 / 7 = 6                   | w_3 = w_1 - q_3*w_2                    | 
|                                    |      = 3 ^ (6*64) = 387                | 
+------------------------------------+----------------------------------------+
| r_4 = r_2 mod r_3                  | v_4 = v_2 - q_4*v_3                    | 
|      = 7 mod 2 = 1                 |      = 63 ^ (3*131) = 442              | 
| q_4 = 7 / 2 = 3                    | w_4 = w_2 - q_4*w_3                    | 
|                                    |      = 64 ^ (3*387) = 709              | 
+------------------------------------+----------------------------------------+
| r_5 = r_3 mod r_4                  | GCD = r_4 = 1                          | 
|      = 2 mod 1 = 0                 | => Nghịch đảo nhân là w_4 = 709        | 
| q_5 = 2 / 1 = 2                    |                                        | 
+------------------------------------+----------------------------------------+
```

Kết luận chung: **$523^{-1} \equiv 798$ và $1015^{-1} \equiv 709$** trong trường hữu hạn đã cho.
